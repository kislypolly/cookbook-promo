Книга рецептов

Удобное приложение для поиска рецептов по названию и ингредиентам.

Посмотреть демо Узнать больше

Возможности приложения

Поиск по названию

Быстрый поиск рецептов по названию блюда.

Поиск по ингредиентам

Находи рецепты из того, что есть в холодильнике.

Детальная карточка рецепта

Ингредиенты, пошаговая инструкция и фото блюда.

Категории блюд

Завтраки, супы, десерты и многое другое.

Избранное

Сохраняй любимые рецепты и возвращайся к ним в один клик.

Удобный интерфейс

Минимизм, крупные фото и понятная навигация.

Преимущества

  • Экономия времени за счёт умного поиска рецептов.
  • Удобная структура: категории, фильтры и избранное.
  • Интерфейс, который приятно использовать каждый день.
